**Dependencies:**
  - Python 3.14.2
  - Pandas 2.3.3

**HOW TO USE:**
  1. add csv file named, "draft_board.csv" with all data removed besides team name in left column, and player name in right column (either empty or has name of retained player) in the following columns:
    PL draft board in columns A & B
    ML draft board in columns C & D
    CL draft board in columns E & F
    AL draft board in columns G & H
    FL draft board in columns I & J
  2. Create an empty directory named, "results" in the root folder
  3. run \_\_main.py\_\_

.txt files will be created for each league in the results directory.
